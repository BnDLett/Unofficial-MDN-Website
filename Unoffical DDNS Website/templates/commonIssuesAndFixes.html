Common issues and fixes

Connection issues

1. If you're getting the error "failed to load world data" or "connection closed." Then this can sometimes be an issue on your end. If this happens then try to connect to the server with a more direct connection. (Using host-name and port) for example: mindustry.ddns.net:1000 (Survival), using this could help possibly fix some issues.

2. If that doesn't work then try changing your DNS server or connecting via IP (92.42.45.96).

3. If the fixes above doesn't fix your issues then please contant someone in our Discord server.

Foo's client issues

1. If you're on the unstable version of Foo's then switch to the stable version. Switching to the stable version has fixed problems before. 2. If the fix above doesn't fix your issues then contact someone in our Discord server.
